# תמונות ולוגו בשקפים לא מופיעות

אם העלית תמונה או לוגו אבל הם לא מוצגים (או מופיע אייקון שבור), סביר שה-bucket של Supabase Storage **לא מוגדר כציבורי**.

## פתרון: להפוך את ה-bucket לציבורי

1. נכנס ל-**Supabase Dashboard** → **Storage**
2. בוחרים את ה-bucket **`slide-images`** (או יוצרים אותו אם לא קיים)
3. לוחצים על **⋮** (שלוש נקודות) ליד ה-bucket → **Edit**
4. מסמנים **"Public bucket"** (bucket ציבורי)
5. שומרים

או:

1. **Storage** → **Policies**
2. מוודאים שיש policy שמאפשרת **SELECT (read)** לכלל המשתמשים על ה-bucket `slide-images`

אחרי ההגדרה, תמונות ולוגואים שמועלים דרך העורך אמורים להופיע בשקפים.
